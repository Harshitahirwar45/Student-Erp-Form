# 🎓 Student ERP Sign Up Form

A simple and responsive **Student ERP Sign Up Form** built using **HTML5** and **CSS3**.

This project was created to practice front-end web development concepts such as HTML forms, responsive layouts, modern UI design, and CSS styling.

---
<img width="1315" height="741" alt="image" src="https://github.com/user-attachments/assets/eacacc4c-f5c7-4afd-9aba-db31a473ac35" />


## 🚀 Features

- Modern card-based UI
- Clean and user-friendly interface
- Student registration form
- Password fields
- Large city selection dropdown
- Form submission page
- Hover effects and input focus animations

---

## 📋 Form Fields

The registration form contains the following fields:

- Full Name
- Father's Name
- Mother's Name
- Address
- Roll Number
- Password
- Confirm Password
- City Selection
- Submit Button

## ▶️ How to Run

1. Download or clone the repository.

```
git clone https://github.com/your-username/student-erp-sign-up.git
```

2. Open the project folder.

3. Open **index.html** in your web browser.

No additional software or installation is required.
